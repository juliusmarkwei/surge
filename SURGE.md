## What SURGE Does

SURGE is a free, open-source Mac system cleaner that helps you reclaim disk space by:

- Removing system and user caches
- Cleaning up log files
- Emptying trash safely
- Removing old downloads
- Cleaning developer junk (Xcode, npm, Homebrew, etc.)

## Key Features

1. **Smart Care** - One-click system optimization
2. **Storage Management** - Category-based cleanup with preview
3. **Safety First** - Multi-layer protection, quarantine system
4. **Native UI** - Beautiful SwiftUI interface
5. **Menu Bar Monitoring** - Real-time system stats
